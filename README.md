## 🧥 Code Convention

### Getting Started

1. `clone` the repository,

```
$ git clone "https://github.com/hasunghwa/hetflix.git"
```

2. `Install` dependencies,

```
$ npm install
```

3. `start` the project,

```
$ npm start
```
